# dev-sather-cli-hello-world

## Description
A POC for sather programming language.

## Tech stack
- g++
- make
- ocaml
- python

## Docker stack
- archlinux
- rzuckerm/sather

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
